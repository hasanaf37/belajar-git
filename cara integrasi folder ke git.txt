Cara integrasi folder ke repository GitHub

1. buka folder nya kemudian klik kanan pilih gitbash
2. di cli ketikkan perintah git init , git add *, git commit -m "first commit" , git branch -M main, git remote add origin https://github.com/hasanaf37/belajar-git.git , git push -u origin main



Catatan : Semisal eror pada saat git push  pastikan local sudah terintegrasi dengan GitHub 
         ketikkan git pull origin main
                                 (nama branch) 

Git pull & Request

1. command menambahkan branch  - >   git checkout -b feat/branch baru      // membuat branch baru 
						        ( nama branch nya )

2. git push origin feat/branchbaru                                         //push branch ke GitHub 

3. klik compare & pull request
